# Using R with GPU's

* [torch for R](https://torch.mlverse.org/) - requires CUDA and cuDNN
* [reticulate](https://rstudio.github.io/reticulate/) - requires CUDA and python
* [cuda.ml](https://github.com/mlverse/cuda.ml) - requires CUDA and RAPIDS
